== Installation ==

**Automatic Installation:**

1. Go to Plugins > Add New
2. Search for "WeTechForU AI Chat Widget"
3. Click "Install Now"
4. Activate the plugin

**Manual Installation:**

1. Download the plugin zip file
2. Go to Plugins > Add New > Upload Plugin
3. Choose the zip file and click "Install Now"
4. Activate the plugin

**Configuration:**

1. Go to Chat Widget in your WordPress admin menu
2. Get your widget key from [WeTechForU.com](https://wetechforu.com)
3. Enter your widget key
4. Enable the widget
5. Click "Save Settings"
6. Test the connection

That's it! The chat widget will now appear on your website.
