API Setup Instructions
Google My Business API (FREE)
  1. Go to Google Cloud Console
  2. Create a new project or select existing
  3. Enable Google My Business API
  4. Create credentials (API Key)
  5. Copy the API key and paste above
Facebook Business API (FREE)
  1. Go to Facebook Developers
  2. Create a new app
  3. Add Business Manager product
  4. Get App ID and App Secret
  5. Copy credentials and paste above
Tip: All APIs work with localhost for development. No domain required for testing!
